# Stabilization of Integral Difference Equations by Solving a Corona Problem

Python code for reproducing the numerical simulations in the article:

**"Stabilization of Integral Difference Equations by Solving a Corona Problem"**

The article will appear in *IEEE Systems & Control Letters*.  
A HAL version of the article is available [here](https://hal.science/hal-05558728).

## Repository contents

- `run_simulation.py`: main script used to run the numerical simulations.
- `parameters.py`: file containing the parameters of the integral difference equation.
- `requirements.txt`: Python packages required to run the code.

## How to use

First install the required Python packages:

```bash
pip install -r requirements.txt

Then run the simulation with:

```bash
python run_simulation.py

To change the example system or numerical parameters, edit:

```text
parameters.py
